# 🚂 Rail Vision 3D

**Система компьютерного зрения для робота — 3D-детекция рельсов**

Полный стек компьютерного зрения, позволяющий роботу видеть и определять рельсы в 3D-пространстве, аналогично человеческому зрению.

## Архитектура

```
Стерео-камера → Калибровка → Глубина (SGBM) → Сегментация (DeepLabV3+) → 3D-реконструкция → Трекинг → Визуализация
```

## Установка

```bash
pip install -r requirements.txt
pip install -e .
```

## Быстрый старт

```bash
# Демо на синтетических данных
python scripts/demo.py

# Калибровка стерео-камеры
python scripts/calibrate.py --left 0 --right 1

# Обучение модели
python scripts/train.py --config config/config.yaml
```

## Структура проекта

| Модуль | Описание |
|--------|----------|
| `camera/` | Стерео-камера, калибровка, оценка глубины |
| `detection/` | Нейросетевая сегментация рельсов (DeepLabV3+, U-Net) |
| `reconstruction/` | 3D облако точек, RANSAC-фитинг |
| `tracking/` | Трекинг рельсов (фильтр Калмана) |
| `visualization/` | 2D/3D визуализация |
| `pipeline/` | Главный конвейер обработки |

## Технологии

- **PyTorch** — нейросетевая сегментация
- **OpenCV** — обработка изображений, стерео-зрение
- **Open3D** — 3D визуализация и обработка облаков точек
- **SciPy** — научные вычисления, сплайны
